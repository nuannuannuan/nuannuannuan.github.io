使用hexo 搭建自己的博客

首先

npm i hexo-cli -g

hexo init myblog

cd myblog

npm i

hexo server

新建完成后修改_config.yml

安装hexo-deployer-git 自动化部署发布工具

npm install hexo-deployer-git –save

发布到github

输入命令hexo clean hexo g hexo d

hexo clean # 清除缓存文件等

hexo g # 生成页面

hexo d # 发布到git

hexo s # 启动预览
