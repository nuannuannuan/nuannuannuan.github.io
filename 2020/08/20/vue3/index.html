vue3的亮点

  1. runtime 比vue2快两倍
    通常vue的模板要编译成render函数去渲染
    实现运行时编译，
    做算法对比的时候不需要所有的进行比对
  2. tree shaking 按需编译代码
    按需打包，用到才会引入，
  3. 更优秀的支持ts
  4. compositionAPI
  5. custom Renderer API (自定义渲染器)
  6. 内置新特性
